# Kiki's Honey Musty

The official showcase site for **Kiki's Honey Musty** — handcrafted, small-batch honey mustard made with love in Eugene, Oregon.

This is a launch/showcase site only. No ordering or e-commerce — just the brand story, where to find us at local pop-ups, and how to get in touch.

## About the site

It's a single self-contained HTML file. The logo is base64-embedded directly into `index.html`, so there are no external image dependencies. Fonts load from Google Fonts.

## Preview locally

Just open `index.html` in any modern browser — no build step, no server required.

```sh
open index.html
```

## Deployment

Intended for **GitHub Pages** with a custom domain at **kikishoneymusty.com**.

To deploy:
1. In this repo, go to **Settings → Pages**
2. Under *Build and deployment*, choose **Deploy from a branch**
3. Select `main` / `root` and save
4. Configure the custom domain (`kikishoneymusty.com`) in the same Pages settings, and point the domain's DNS at GitHub Pages
